Control and monitor Debug Unit I/O.

DAP Debug Input Output commands control and monitor Debug Unit Outputs/Inputs (Digital/Analog). This command can be used to control and monitor Debug Unit Inputs/Outputs. Debug host typically controls LED indicators:

  • Debugger Connected and
  • Target Running

Command to write data to a signal.

BYTE | BYTE **| WORD **|
> 0x01 | Signal | Output |
******|********|////////|


Response to a write command.

BYTE | BYTE **| WORD **|
< 0x01 | Status | Input |
******|********|////////|


Where

  • Signal: Selects signal and I/O operation
    • Bit6..0: Signal
      • 0 - "Debugger Connected" indicator (LED)
      • 1 - "Target Running" indicator (LED)
      • ... Custom I/O (unassigned by CMSIS-DAP)
    • Bit7: I/O operation
      • 0 - Write
      • 1 - Read
  • Output: Value written to selected signal
  • Input: Value read from selected signal
